= Getting Started



== Installation


1. Getting Started

```shell
django-admin startproject <project_name>
```

2. Running


```shell
python manage.py runserver
```

== Route


Create new app

```
django-admin startapp home
```


user -> `/home` -> link:./saas_django/urls.py[`url.py`]'s url pattern match with `home` -> link:./home/views.py[`views.py`] -> `def home`

It is MVT Framework.

Model -> View -> Template
